# Security

Report security issues for this repository through a private channel.

- Do not create public GitHub issues for security-sensitive vulnerabilities.
- If this repository has [GitHub Security Advisories](https://docs.github.com/en/code-security/security-advisories) enabled, use the `Report a vulnerability` flow on the repository.
- If Security Advisories are unavailable, use the repository owner-maintained private disclosure mechanism.

Include:

- scanner output excerpt (redacted)
- repository SHA or tag
- commit/branch used
- workflow context and reproduction steps
